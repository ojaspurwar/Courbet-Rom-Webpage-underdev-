Installation Guide

Follow these simple steps to flash Courbet ROM safely.

01

Unlock Bootloader

Ensure your device's bootloader is unlocked. This is essential for installing any custom software.

02

Flash Recovery

Install a compatible custom recovery like TWRP or OrangeFox via Fastboot mode.

fastboot flash recovery recovery.img
03

Wipe Data

Boot into recovery and perform a clean flash. Format Data is recommended (Type 'yes').

04

Flash ROM

Locate the Courbet ROM zip file and flash it. Reboot your device and enjoy!
