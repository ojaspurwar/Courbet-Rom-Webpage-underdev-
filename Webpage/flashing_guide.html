Clean Flash

REQUIRED WHEN: Coming from a different ROM (e.g., EvolutionX to LineageOS), updating Android versions (A15 ‚Üí A16), or major updates. Erases user data.
  1. Boot into OrangeFox Recovery.
  2. Tap the Trash Icon -> Format Data -> Type yes to confirm.
  3. Go to Menu -> Install -> Select ROM Zip -> Swipe to Flash.
  4. (Optional) Flash GApps package without rebooting if required.
  5. Return to main menu -> Reboot -> System.
5

Dirty Flash (Update)

ONLY FOR: Updating the SAME ROM to a newer version on the SAME Android version. Keeps your data safe.
  1. Boot into OrangeFox Recovery.
  2. Select Install and flash the updated ROM ZIP file.
  3. (Optional) Re-flash GApps/Magisk if not automatically restored.
  4. Wipe the Dalvik/ART Cache.
  5. Reboot to System.
